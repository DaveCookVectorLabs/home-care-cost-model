Case Studies
============

Three worked examples from the working paper (Section IX).

Case 1 — Mrs. A., 82, Sudbury ON, mild frailty
----------------------------------------------

Katz ADL 5, Lawton IADL 5, intact cognition, cane, frailty, lives alone.
Daughter provides 5 hours/week of informal caregiving. Demonstrates the
"cleaner is enough" archetype: subsidised program covers most personal
support hours; out-of-pocket is predominantly housekeeping.

Case 2 — Mr. B., 74, Vancouver BC, moderate dementia, post-stroke
-----------------------------------------------------------------

Katz ADL 2, Lawton IADL 1, moderate cognition, walker, dementia, lives
with spouse. Household net income $72,000, DTC certificate. Demonstrates
the "hybrid beats all-PSW" archetype: cost-minimising service mix saves
roughly $436/month versus all-PSW plan; tax credits reduce out-of-pocket
by $7,670/year.

Case 3 — Mr. C., 89, rural NB, bedbound veteran
-----------------------------------------------

Katz ADL 0, Lawton IADL 0, severe cognition, bedbound, lives with
daughter. Demonstrates the "VIP + DTC + CCC + METC stacking" archetype:
the full tax-and-benefit stack reduces what would otherwise be a
$10,800/month cost to roughly $3,575/month, making aging in place
financially viable at the ceiling of what the model supports.

Each case is reproducible via the engine by loading the corresponding
inputs; the reference parameters are documented in the Python engine
test suite at ``engines/python/test_engine.py``.
